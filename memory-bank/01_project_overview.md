# プロジェクト概要

このプロジェクトは、Paradox Interactive社のゲーム「Europa Universalis IV」において、日本語などの2バイト文字を正しく表示させるためのDLL（ダイナミックリンクライブラリ）です。

## 主な機能

-   ゲーム内のテキスト描画処理に介入し、2バイト文字のレンダリングを可能にします。
-   ロギングにD言語標準の`std.logger`を使用し、より統一されたログ管理を実現しています。
-   YAMLファイルのパース処理を強化し、`l_english:`キーの自動補正などにより堅牢性を高めています。
-   バイトパターンマッチング機能を拡張し、特定のアドレスを効率的に取得できるようになりました。

## プロジェクトの目的

元々C++で書かれておりWindowsでしか動作しないこのDLLを、Linux環境でも動作するようにし、D言語で書き直すことを目指しています。D言語への移行は現在も進行中です。

## EU4の翻訳ファイル読み込みとEU4dllの役割

EU4dll自体は翻訳ファイル（.yml）を直接読み込む機能はありません。翻訳ファイルはEU4標準のMOD機能を通じてランチャーによって読み込まれ、ゲームのメモリ上にロードされます。
